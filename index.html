Auto

Usar background-size para redimensionar, o valor padrão é "Auto".
Vai se ajustar automaticamente, pode cortar boa parte da imagem

Cover

A imagem vai cobrir toda a div centralizado. Também pode cortar a imagem.

Contain

Redimensiona a imagem para entrar compeltamente, como a imagem é um retangulo e
o container é um retangulo em pé, a imagem vai se repetir.

Contain retangulo correto

Pelo fato da div ser mais ou menos do formato da imagem, a imagem se repetiu menos.

Valor

Pode se colocar um valor especifico de medida. Se não preencher completamente, a imagem irá se repetir.
Pode colocar valor como percentual. Lembrando que apenas um valor, será considerado apenas o width.
O segundo valor é o Height.

Valor retangulo correto

Colocando "Valor" e 100% para as duas dimenões,
a imagem fica perfeitamente encaixada.
